歡迎來到 EmbyHack 網頁

本頁會指導您在 iOS 如何啟用 Emby Premiere 資格。

Download root credentials
  1. 點擊上方「Download root credentials」按鈕下載憑證。
  2. 到系統設定中，安裝並信任此憑證 (設定 > 一般 > 關於本機 > 憑證信任設定)。
  3. 安裝 Shadowrocket (或其他支援 MitM 的代理軟體)。
  4. 在 Shadowrocket 下方欄位中移動到「配置」。
  5. 點擊 default.conf -> "編輯配置" -> Hosts。
  6. 點擊右上角 + 新增後儲存：
    • 域名：mb3admin.com
    • IP Address (本地值)：本站 IP 位址 (請填入此網頁的伺服器 IP)
  7. 回到 Shadowrocket 首頁，將「未連線」右邊滑塊開啟連接。
  8. 打開 Emby 客戶端，會跳出一視窗，點擊信任即可啟用。
  9. 後續可回到設定中，刪除安裝的憑證，不會影響到當前的授權。

Welcome to EmbyHack

This page will guide you on how to enable Emby Premiere on iOS.

Download root credentials
  1. Click the "Download root credentials" button above.
  2. Go to System Settings, install, and trust this certificate (Settings > General > About > Certificate Trust Settings).
  3. Install Shadowrocket (or other proxy apps supporting MitM).
  4. Go to the "Config" tab at the bottom of Shadowrocket.
  5. Tap default.conf -> "Edit Config" -> Hosts.
  6. Tap "+" at the top right to add and save:
    • Domain: mb3admin.com
    • IP Address: This site's IP address (Enter the IP of this server)
  7. Go back to the Home tab and toggle the switch to connect.
  8. Open the Emby client; a window will pop up. Click "Trust" to enable.
  9. You can go back to settings and delete the installed certificate later; it will not affect the current authorization.
